To run the WebGL example you must run a local webserver.

The easiest way to do this is with Python.

You can get Python from: http://www.python.org/download/

You can start a local webserver with: python -m http.server 8000

You can start the webpage then with: "start http://localhost:8000/TextureMappingWebGL.html"

from the console or batch file.

The best browser to use and debug WebGL apps is Google Chrome.
Open the developer tools from the tools menu of the Chrome browser. These tools provide exelent debugging features and error messages.
For more information on WebGL and supportin browser visit: http://www.khronos.org/webgl/wiki/Main_Page
